# Campus Docs Assistant - 重构版

> 校园AI助手 - AI产品经理职业能力展示平台（好铁汁出品）

## 项目简介

本项目是面向大二学生的AI产品经理职业能力展示平台，集学习工具与职业展示于一体。通过AI产品案例分析、PRD模板库、用户画像构建工具、产品原型设计等核心模块，帮助用户在学习产品管理知识的同时，培养AI产品经理岗位所需的核心职业技能。

## 核心功能

### AI产品案例分析
- 10+不同领域AI产品案例库
- 标准化分析模板（案例背景、功能拆解、AI技术应用、商业模式、优缺点评估、改进建议）
- 搜索、分类筛选、收藏与笔记功能

### PRD模板库
- 3套差异化PRD模板（AI功能类、工具类、服务类）
- 完整文档结构（产品概述、用户需求、功能需求、非功能需求、原型说明、数据指标、风险评估）
- PRD在线编辑功能

### 用户画像构建工具
- 标准化用户画像模板（基本属性、行为特征、需求痛点、用户目标、使用场景）
- 智能标签推荐与自定义标签
- 可视化展示与报告导出

### 产品原型设计基础
- 原型设计方法论知识库
- 基础组件库说明文档
- 简易原型草图工具

## 技术栈

- **前端**: Vue3 + Vite + Element Plus + Pinia
- **后端**: Express + SQLite + FTS5全文检索
- **存储**: IndexedDB本地存储 + 云端备份

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发环境
npm run dev

# 构建生产版本
npm run build
```
